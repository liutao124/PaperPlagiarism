上述参数设置中，参数的含义如下：
对比文件并发数：可以同时对比的文件数量，和系统的CPU、内存、硬盘的读写速度相关，一般设定为10效果比较佳
重复率限定百分比：两个文档重复达到百分值为多少，定义为重复
初始目录：选择对比目录在打开文件夹选择窗口的初始目录
